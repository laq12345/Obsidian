## 表 7.4：Python 内置字符串方法

| 方法                        | 描述                                                    |
| ------------------------- | ----------------------------------------------------- |
| **count**                 | 返回子串在字符串中**不重叠出现**的次数                                 |
| **endswith**              | 如果字符串以指定**后缀**结尾，返回 `True`                            |
| **startswith**            | 如果字符串以指定**前缀**开头，返回 `True`                            |
| **join**                  | 使用字符串作为**分隔符**，将其他字符串序列**拼接**为一个字符串                   |
| **index**                 | 返回子串**首次出现**的起始索引；如果未找到则**抛出 `ValueError` 异常**        |
| **find**                  | 返回子串**首次出现**的首字符位置；与 `index` 类似，但**未找到时返回 `-1`**      |
| **rfind**                 | 返回子串**最后一次出现**的首字符位置；未找到时返回 `-1`                      |
| **replace**               | 将字符串中的匹配项**替换**为另一个字符串                                |
| **strip, rstrip, lstrip** | **去除空白字符**（包括换行符），分别作用于**两侧 / 右侧 / 左侧**               |
| **split**                 | 使用指定的分隔符将字符串**拆分**为子串列表                               |
| **lower**                 | 将字母字符转换为**小写**                                        |
| **upper**                 | 将字母字符转换为大写**大写**                                      |
| **casefold**              | 将字符转换为小写，并将特定区域的变体字符组合转换为统一的可比较形式（比 `lower` 更彻底）      |
| **ljust, rjust**          | 分别进行**左对齐 / 右对齐**；用空格（或其他填充字符）填充字符串的**相反侧**，使结果达到最小宽度 |

---

### 补充说明几个易混淆的方法：

```python
s = "  Hello World  "

# strip 系列
s.strip()    # "Hello World"   → 去两边空格
s.lstrip()   # "Hello World  " → 去左边空格
s.rstrip()   # "  Hello World" → 去右边空格

# casefold vs lower（处理特殊字符时有区别）
"ß".lower()     # "ß"        → 德语 sharp s 不变
"ß".casefold()  # "ss"       # 转为 ss（更彻底的国际化处理）

# ljust / rjust 对齐
"左".ljust(10)   # "左         "  → 左对齐，右侧补空格到宽度10
"右".rjust(10)   # "         右"  → 右对齐，左侧补空格到宽度10
"中".center(10)  # "    中     "  → 居中对齐
```
