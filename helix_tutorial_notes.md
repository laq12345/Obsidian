# Helix 编辑器教程笔记

> Helix 是一个后现代模态文本编辑器（A post-modern modal text editor），受 Vim/Kakoune 启发。

---

## 第一章：基础操作

### 1.1 光标移动
| 按键 | 功能 |
|:---:|:---|
| `h` | 左移 |
| `j` | 下移（形似向下箭头） |
| `k` | 上移 |
| `l` | 右移 |

> 💡 方向键也可用，但 `hjkl` 更高效，因为它们靠近其他常用键。

### 1.2 退出 Helix
| 命令 | 功能 |
|:---:|:---|
| `:q` / `:quit` | 退出（有未保存更改时会失败） |
| `:q!` / `:quit!` | 强制退出，丢弃更改 |
| `Esc` | 取消命令模式 |

### 1.3 删除
| 按键 | 功能 |
|:---:|:---|
| `d` | 删除光标下的字符（或当前选中的内容） |

### 1.4 插入模式
| 按键 | 功能 |
|:---:|:---|
| `i` | 在光标前进入插入模式 |
| `Esc` | 退出插入模式，返回普通模式 |

> 状态栏会显示当前模式：`NOR`（普通）→ `INS`（插入）

### 1.5 保存文件
| 命令 | 功能 |
|:---:|:---|
| `:w` / `:write` | 保存文件 |
| `:wq` / `:write-quit` | 保存并退出 |

> 未保存时，文件名旁会显示 `[+]`。

## 第二章：更多插入命令

| 按键 | 功能 |
|:---:|:---|
| `i` | 在选区前插入 |
| `a` | 在选区后追加（append） |
| `I` | 在行首插入（第一个非空白字符） |
| `A` | 在行尾插入 |
| `o` | 在下方新开一行并插入 |
| `O` | 在上方新开一行并插入 |

## 第三章：移动与选区

### 3.1 基础移动（选择文本）
| 按键 | 功能 |
|:---:|:---|
| `w` | 向前选中到下一个词的开头 |
| `e` | 向前选中到当前词的结尾 |
| `b` | 向后选中到当前词的开头 |

> `d` 删除的是**整个选区**，不只是光标下的字符。

### 3.2 WORDS vs words
| 按键 | 功能 |
|:---:|:---|
| `W/E/B` | 以**空白**为分隔符移动（WORDS） |
| `w/e/b` | 以标点+空白为分隔符移动（words） |

> 例：`one-of-a-kind` 用 `w` 需7次，用 `W` 只需1次。

### 3.3 修改命令
| 按键 | 功能 |
|:---:|:---|
| `c` | 删除选区并进入插入模式（change，相当于 `di`） |

### 3.4 计数前缀
| 用法 | 功能 |
|:---:|:---|
| `2w` | 向前移动2个词 |
| `3e` | 向前移动到第3个词的结尾 |
| `2b` | 向后移动2个词 |

### 3.5 选择模式（Select Mode）
| 按键 | 功能 |
|:---:|:---|
| `v` | 进入选择模式（移动会扩展选区而非替换） |
| `v` / `Esc` | 退出选择模式 |

### 3.6 选择整行
| 按键 | 功能 |
|:---:|:---|
| `x` | 选择当前整行，再次按选择下一行 |
| `2x` | 选择2行 |
| `X` | 选择当前行（不扩展到下一行） |

### 3.7 折叠选区
| 按键 | 功能 |
|:---:|:---|
| `;` | 将选区折叠为单光标（取消选择但保留位置） |
| `Alt-;` | 翻转选区方向（交换光标和锚点） |

## 第四章：撤销、复制与搜索

### 4.1 撤销与重做
| 按键 | 功能 |
|:---:|:---|
| `u` | 撤销 |
| `U` | 重做 |

### 4.2 复制与粘贴
| 按键 | 功能 |
|:---:|:---|
| `y` | 复制（yank）选区 |
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |
| `Space + y` | 复制到系统剪贴板 |
| `Space + p` | 从系统剪贴板粘贴 |

> ⚠️ 删除或修改的文本会自动复制到剪贴板。用 `Alt-d` / `Alt-c` 可避免此行为。

### 4.3 搜索
| 按键 | 功能 |
|:---:|:---|
| `/` | 向前搜索 |
| `?` | 向后搜索 |
| `Enter` | 确认搜索 |
| `n` | 跳到下一个匹配（始终向前） |
| `N` | 跳到上一个匹配（始终向后） |

> 支持正则表达式。

### 5.1 多光标

### 5.3 对齐选区
| 按键 | 功能 |

| `&` | 对齐所有选区的"头部"（移动端） |

### 5.4 按行分割选区

|:---:|:---|
| `Alt-s` | 在换行符处分割选区 |

## 第六章：字符查找与重复

### 6.1 查找字符
| 按键 | 功能 |
|:---:|:---|
| `f<char>` | 选到并包含某字符（find） |
| `t<char>` | 选到但不包含某字符（till） |
| `F<char>` | 向后查找并包含 |
| `T<char>` | 向后查找但不包含 |

> 不同于 Vim，Helix 的 `f/t` 不限于当前行，会在整个文件中搜索。

### 6.2 替换字符
| 按键 | 功能 |
|:---:|:---|
| `r<char>` | 将选中的所有字符替换为指定字符 |

### 6.3 重复命令
| 按键 | 功能 |
|:---:|:---|
| `.` | 重复上一次的插入操作 |
| `Alt-.` | 重复上一次的 `f/t` 选择 |

| 按键 | 功能 |
|:---:|:---|

## 第十二章：匹配模式（Match Mode）

按 `m` 进入匹配模式，显示可用操作弹窗。

| 命令 | 功能 |
|:---:|:---|
| `mm` | 跳转到匹配的括号/符号 |
| `mi(` / `mi)` | 选择括号**内部**内容（不含括号） |
| `ma(` / `ma)` | 选择括号**周围**内容（含括号） |
| `ms(` / `ms)` | 为选区添加括号包围 |
| `md(` / `md)` | 删除周围的括号对 |
| `mr([` | 将周围 `()` 替换为 `[]` |

> 支持 `()`, `[]`, `{}`, `""`, `''` 等多种分隔符，可处理嵌套结构。

## 第十三章：分屏（Splits）

### 13.1 创建分屏
| 命令/按键 | 功能 |
|:---:|:---|
| `Ctrl-w n v` | 垂直分屏（右侧新建空缓冲区） |
| `Ctrl-w n s` | 水平分屏（下方新建空缓冲区） |
| `:vs [文件名]` | 垂直分屏打开文件 |
| `:hs [文件名]` | 水平分屏打开文件 |

### 13.2 移动与关闭
| 按键 | 功能 |
|:---:|:---|
| `Ctrl-w h/j/k/l` | 向左/下/上/右移动 |
| `Ctrl-w w` | 切换到下一个分屏 |
| `Ctrl-w q` | 关闭当前分屏 |
| `Ctrl-w o` | 关闭其他所有分屏，只保留当前 |

### 13.3 当前缓冲区分屏
| 按键 | 功能 |
|:---:|:---|
| `Ctrl-w s` | 水平分割当前缓冲区 |
| `Ctrl-w v` | 垂直分割当前缓冲区 |

### 13.4 交换与转置
| 按键 | 功能 |
|:---:|:---|
| `Ctrl-w H/J/K/L` | 与左/下/上/右的分屏交换位置 |
| `Ctrl-w t` | 转置分屏方向（横↔竖） |

### 13.5 从文件选择器打开
| 按键 | 功能 |
|:---:|:---|
| `Space f` | 打开文件选择器 |
| `Ctrl-v` | 在垂直分屏中打开选中文件 |
| `Ctrl-s` | 在水平分屏中打开选中文件 |
| `Esc` | 退出文件选择器 |

## 核心设计哲学

1. **选区优先（Selection-oriented）**：操作基于选区，而非单个光标
2. **多光标原生支持**：`C` / `s` / `Alt-s` 等命令天然支持多位置编辑
3. **组合命令**：数字前缀 + 移动 + 操作（如 `3wd`）
4. **模态编辑**：普通模式导航，插入模式输入，命令模式执行
