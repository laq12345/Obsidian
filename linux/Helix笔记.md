# Helix 编辑器学习笔记

> 基于 Helix 内置教程整理，涵盖从入门到高级的常用操作。

---

## 引言

Helix 是一款**模态**编辑器，主要模式为 **Normal（普通）** 与 **Insert（插入）**。
在 Normal 模式下，按键不会直接输入文本，而是执行各类文本操作，从而实现高效编辑。
本教程将逐步带你掌握 Helix 的模态编辑特性。
开始前，请确认 CapsLock 已关闭，然后按住 `j` 键滚动至第一课。

---

## 第1章：基础操作

### 1.1 基本光标移动

使用 `h`, `j`, `k`, `l` 键移动光标：

```
        ↑
        k
  ← h       l →
        j
        ↓
```

- `h` 左移
- `l` 右移
- `j` 下移（形似向下箭头）
- `k` 上移

> 方向键也可用，但 `hjkl` 更靠近其他常用键，效率更高。

### 1.2 退出 Helix

1. 输入 `:` 进入命令模式（光标会移至屏幕底部）
2. 输入 `q` 或 `quit` 并回车，退出 Helix。

- 若有未保存更改，需使用 `q!` 或 `quit!` 强制退出并**丢弃**改动。
- 按 `Escape` 可取消命令。

### 1.3 删除字符

在 Normal 模式下按 `d` 键，删除光标所在字符。
重复操作即可删除多余字符。

### 1.4 插入模式

- 按 `i` 进入 Insert 模式，此后按键正常输入文本。
- 按 `Escape` 返回 Normal 模式。
- 底部状态栏会显示当前模式：`NOR`（Normal）或 `INS`（Insert）。

### 1.5 保存文件

- `:w` 或 `:write` 保存文件。
- `:wq` 或 `:write-quit` 保存并退出。
- 可指定路径：`:w 路径/文件名`。
- 状态栏文件名旁出现 `[+]` 表示有未保存更改。

### 第1章 总结

| 操作 | 功能 |
|------|------|
| `h/j/k/l` | 移动光标 |
| `:` | 进入命令模式 |
| `:q` / `:q!` | 退出/强制退出 |
| `:w` | 保存 |
| `:wq` | 保存并退出 |
| `d` | 删除光标处字符 |
| `i` | 进入插入模式 |
| `Esc` | 返回普通模式 |

## 第2章：更多插入方式

### 2.1 其他插入命令

| 命令 | 功能 |
|------|------|
| `i` | 在光标前插入 |
| `a` | 在光标后插入（append） |
| `I` | 在行首插入（第一个非空白字符前） |
| `A` | 在行尾插入 |

### 2.2 新起一行

- `o`：在光标下方新建行并进入插入模式。
- `O`：在光标上方新建行并进入插入模式。

## 第3章：移动与选择

### 3.1 词移动与选择

- `d` 实际是**删除选中文本**；默认光标就是单个字符的选区。
- `w` 向前选中到下一单词开头，然后按 `d` 即可删除该词。

### 3.2 更多移动命令

| 命令 | 功能 |
|------|------|
| `w` | 向前移动到下一单词开头 |
| `e` | 向前移动到当前单词末尾 |
| `b` | 向后移动到当前单词开头 |
| `W/E/B` | 同上，但以空白字符作为单词边界（WORD） |

示例：`one-of-a-kind` 视为一个 WORD，用 `W` 一个键即可选中。

### 3.3 更改命令 `c`

- `c` 删除当前选区并进入插入模式（等同于 `di`）。
- 常用流程：`e` 选中单词 → `c` 删除并输入新词。

### 3.4 计数移动

在移动命令前加数字，重复指定次数。
例：`2w` 前移两个单词，`3e` 移动到第三个单词末尾，`2b` 后移两个单词。

### 3.5 选择模式（Select mode）

- `v` 进入选择模式，此后任何移动都会**扩展**选区。
- 再次 `v` 或 `Esc` 返回 Normal 模式。
- 在选择模式下，移动命令不会替换选区，而是扩大选区。

### 3.6 选择整行

- `x` 选中当前整行，再按一次 `x` 继续选中下一行（类似 `X` 不过 `X` 对空行无效）。
- 常用组合：`x` 选中行 → `d` 删除。

### 3.7 折叠选区

- `;` 将选中的多光标/选区折叠为单个光标（取消选区）。
- `Alt-;` 翻转选区方向（交换光标与锚点）。

### 第3章 总结

| 命令 | 功能 |
|------|------|
| `w/e/b` | 按单词移动/选择 |
| `W/E/B` | 按 WORD 移动 |
| `d` | 删除选中文本 |
| `c` | 删除选中文本并进入插入模式 |
| `数字` + 移动 | 重复移动 |
| `v` | 进入选择模式（扩展选区） |
| `x` | 选中整行 |
| `;` | 折叠选区 |

## 第4章：撤销、复制粘贴与搜索

### 4.1 撤销与重做

- `u` 撤销。
- `U` 重做。

### 4.2 复制与粘贴

- `y` 复制（yank）选中文本。
- `p` 粘贴到光标后。
- `P` 粘贴到光标前。
- 注意：删除或修改操作会自动复制被修改文本，用 `Alt-d` / `Alt-c` 可避免。
- **系统剪贴板**：默认不共享，使用 `Space + y` / `Space + p` 操作系统剪贴板。

### 4.3 文件内搜索

- `/` 向前搜索，回车确认。
- `?` 向后搜索（但 `n`/`N` 的方向不变：`n` 始终向“后”一个匹配，`N` 向前一个匹配）。
- 支持正则表达式。

### 第4章 总结

| 命令 | 功能 |
|------|------|
| `u` / `U` | 撤销/重做 |
| `y` / `p` | 复制/粘贴 |
| `Space y` / `Space p` | 系统剪贴板复制/粘贴 |
| `/` / `?` | 搜索（正则） |
| `n` / `N` | 跳转至下一/上一匹配 |

## 第5章：多光标与高级选择

### 5.1 多光标

- `C` 在下一个合适的行复制光标（跳过空行或短行）。
- `Alt-C` 在上一个合适的行复制光标。
- 多光标下，编辑指令会同时作用于所有光标。
- `,` 移除主光标（第一个光标）。

### 5.2 选择命令 `s`

- 选中一段文本后按 `s`，输入正则表达式，回车后所有匹配项将被多光标选中。
- 随后可用 `c` 统一修改。
- 例：选中整行 → `s` → 输入 `apples` → 全部选中 → `c` 改为 `oranges`。

### 5.3 正则选择

- `s` 支持正则。例如选择 `'  +'` 可选中多个连续空格，再 `c` 替换为单个空格。
- **文件级查找替换**：`%` 选中全文 → `s` 输入正则 → `c` 修改。

### 5.4 对齐选区

- `&` 对齐选区的“头部”（移动端），使内容整齐排列。
- 常与多光标配合使用：`C` 复制光标 → 选中需对齐部分 → `&`。

### 5.5 按行分割选区

- `Alt-s` 将跨行选区拆分为每行独立选区，便于后续对齐等操作。

### 第5章 总结

| 命令 | 功能 |
|------|------|
| `C` / `Alt-C` | 向下/上复制光标 |
| `s` | 选中选区内的正则匹配 |
| `&` | 对齐选区 |
| `Alt-s` | 按新行分割选区 |
| `,` | 移除主光标 |

## 第6章：字符跳选与重复

### 6.1 跳至指定字符

- `f<字符>` 向前选中到**包含**该字符的位置（find）。
- `t<字符>` 向前选中到**不包含**该字符的位置（till）。
- `F` / `T` 向后执行同样操作。
- Helix 不限于当前行，会在**整个文件**中搜索字符。

### 6.2 替换字符

- `r<字符>` 将选中文本的每个字符替换为指定字符。
- 例：选中分隔符 `=`，`r-` 替换为 `-`。

### 6.3 重复操作

- `.` 重复上一次插入操作。
- `Alt-.` 重复上一次 `f` / `t` 选择。

### 第6章 总结

| 命令 | 功能 |
|------|------|
| `f` / `t` | 向前查找/直到字符 |
| `F` / `T` | 向后查找/直到字符 |
| `r` | 替换选中字符 |
| `.` | 重复上次插入 |
| `Alt-.` | 重复上次 f/t 选择 |

## 第7章：寄存器、连接与缩进

### 7.1 用复制内容替换

- `R` 用先前复制的文本替换当前选中文本。
- 例：`y` 复制 "watermelons" → 选中 "oranges" → `R` 替换。

### 7.2 连接行

- `J` 将选中的多行连接为一行。

### 7.3 缩进

- `>` 增加缩进（向右）。
- `<` 减少缩进（向左）。

### 7.4 增减数字

- `Ctrl-a` 将选中数字加 1。
- `Ctrl-x` 将选中数字减 1。

### 第7章 总结

| 命令 | 功能 |
|------|------|
| `R` | 用复制文本替换 |
| `J` | 连接行 |
| `>` / `<` | 增加/减少缩进 |
| `Ctrl-a` / `Ctrl-x` | 数字递增/递减 |

## 第8章：寄存器与宏

### 8.1 寄存器

- 寄存器是存放文本、搜索词、宏等内容的“容器”，用单个字符命名。
- `"<字符>` 选择指定寄存器。例如 `"b` 选择寄存器 `b`。
- 随后执行 `y` / `p` 等操作即针对该寄存器。

### 8.2 宏

- `Q` 开始/停止录制宏（默认寄存器 `@`）。
- 录制时底部会有提示。
- 停止录制后，按 `q` 重放宏（可从 `@` 或指定寄存器重放）。

### 第8章 总结

| 命令 | 功能 |
|------|------|
| `"<char>` | 选择寄存器 |
| `Q` | 录制/停止宏 |
| `q` | 播放宏 |

## 第9章：搜索选区与跳转

### 9.1 搜索当前选中词

- `*` 将当前主选区的文本复制到搜索寄存器 `/` 中，相当于设置搜索词。
- 随后可用 `n` / `N` 跳转。

> 原理：`*` 实际是 `"/y` 的快捷方式。

### 9.2 选择模式下的搜索

- 在 Select 模式下，`n` / `N` **不会移动选区，而是为每个匹配新增一个光标**。
- 常用流程：`*` 设置搜索词 → `v` 进入选择模式 → `n` 产生多光标 → `c` 统一修改。

### 9.3 跳转列表（jumplist）

- 大范围移动（如搜索）会被自动记录。
- `Ctrl-s` 手动保存当前位置到跳转列表。
- `Ctrl-i` 向前跳转（进），`Ctrl-o` 向后跳转（出）。

### 9.4 双字符标签跳转

- `gw` 启用双字符标签，当前可见区域的每个单词开头会显示两个高亮字符。
- 输入对应两个字符即可跳转到该位置；`Esc` 取消标签。

### 第9章 总结

| 命令 | 功能 |
|------|------|
| `*` | 将主选区设为搜索词 |
| `n`/`N` (选择模式) | 为每个匹配添加光标 |
| `Ctrl-s` | 保存到跳转列表 |
| `Ctrl-i` / `Ctrl-o` | 跳转列表前进/后退 |
| `gw` | 双字符标签跳转 |

## 第10章：选区管理

### 10.1 循环主选区

- `)` 向后循环主选区，`(` 向前循环主选区。
- `Alt-,` 移除主选区（删除最前面那个光标）。

### 10.2 循环选区内容

- `Alt-)` / `Alt-(` 在多选区间循环选中的文本内容。
- 常用于 `s` 匹配后，在多个候选中切换。

### 10.3 改变大小写

- `~` 翻转选中字母的大小写。
- `` ` `` 将选中字母转为小写。
- `Alt-`` 将选中字母转为大写。

### 10.4 按模式分割选区

- `S` 按正则模式分割选区。
- 例：选中句子 → `S` → 输入 `\. |! `（注意空格）→ 按句子分割 → 配合其他命令处理。

### 第10章 总结

| 命令 | 功能 |
|------|------|
| `)` / `(` | 循环主选区 |
| `Alt-,` | 移除主选区 |
| `Alt-)` / `Alt-(` | 循环选区内容 |
| `~` / `` ` `` / `Alt-`` | 大小写转换 |
| `S` | 按正则分割选区 |

## 第11章：注释

- `Ctrl-c` 注释/取消注释当前行（取决于语言注释规则）。
- 若存在多光标或选区，可以一次性注释多行。
- 已经被注释的行会再次被注释（即不会智能反注释）。

## 第12章：匹配模式（match mode）

在 Normal 模式按 `m` 进入匹配模式，屏幕显示可用操作。

### 12.1 跳转至匹配括号

- 光标位于括号 `( )`、`[ ]`、`{ }` 等之一上时，按 `mm` 跳至与其匹配的另一半。

### 12.2 选中内部内容（inside）

- `mi(` 或 `mi)` 选中括号内的内容（不包含括号本身）。
- 适用于 `[]`、`{}`、`""` 等。

### 12.3 选中外部内容（around）

- `ma(` 或 `ma)` 选中括号及其内部内容（包含分隔符）。

### 12.4 添加包围（surround）

- 先选中文本，然后 `ms(` 或 `ms)` 用括号包裹选中文本。
- 同样支持 `ms'`、`ms"`、`ms[` 等。

### 12.5 删除包围

- 光标位于括号内部任意位置，按 `md(` 或 `md)` 删除最近的一对括号。
- 支持嵌套，删除最内层匹配。

### 12.6 替换包围

- `mr([` 将当前最近的括号对（如 `()`）替换为另一对（如 `[]`）。

### 第12章 总结

| 命令 | 功能 |
|------|------|
| `m` | 进入匹配模式 |
| `mm` | 跳至匹配括号 |
| `mi` | 选中括号内部 |
| `ma` | 选中括号及内部 |
| `ms` | 为选中文本添加包围 |
| `md` | 删除包围 |
| `mr` | 替换包围 |

## 第13章：分屏操作

### 13.1 创建分屏

- `Ctrl-w nv` 创建一个垂直分屏（右侧新空白缓冲区）。
- `Ctrl-w ns` 创建一个水平分屏（下方新空白缓冲区）。

### 13.2 在分屏间移动

- `Ctrl-w h/j/k/l` 移动到左/下/上/右分屏。
- `Ctrl-w w` 移至下一个分屏（按打开顺序）。
- `Ctrl-w q` 关闭当前分屏。
- `Ctrl-w o` 关闭除当前外的所有分屏。

### 13.3 查看同一缓冲区的分屏

- `Ctrl-w s` 水平分割当前缓冲区视图。
- `Ctrl-w v` 垂直分割当前缓冲区视图。

### 13.4 使用命令分屏

- `:vsplit 文件名` 或 `:vs 文件名` 垂直分屏并打开文件（若无对应文件则新建缓冲区）。
- `:hsplit 文件名` 或 `:hs 文件名` 水平分屏并打开文件。

### 13.5 交换分屏

- 在目标分屏中按 `Ctrl-w K`（大写）与上方分屏交换位置。
- 同理 `Ctrl-w H/J/L` 分别与左/下/右分屏交换。

### 13.6 转置分屏方向

- `Ctrl-w t` 将当前窗口的垂直分屏转置为水平，再次按下恢复。
- 每按一次，均基于当前窗口的子分屏进行转置。

### 13.7 从文件选择器打开分屏

- `Space f` 打开文件选择器（模糊搜索）。
- 选中文件后：
  - `Ctrl-v` 在垂直分屏打开。
  - `Ctrl-s` 在水平分屏打开。
- `Esc` 退出选择器。
