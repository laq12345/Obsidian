## 1. 光标移动

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + A` | 移到行首 |
| `Ctrl + E` | 移到行尾 |
| `Ctrl + B` | 左移一个字符（同 `←`） |
| `Ctrl + F` | 右移一个字符（同 `→`） |
| `Alt + B` | 左移一个单词 |
| `Alt + F` | 右移一个单词 |
| `Ctrl + XX` | 光标在行首和当前位置间切换 |

---

## 2. 文本编辑

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + D` | 删除光标处字符（同 Delete） |
| `Ctrl + H` | 删除前一个字符（同 Backspace） |
| `Ctrl + W` | 删除前一个单词 |
| `Alt + D` | 删除后一个单词 |
| `Ctrl + U` | 删除光标到行首 |
| `Ctrl + K` | 删除光标到行尾 |
| `Ctrl + Y` | 粘贴刚才删除的内容（yank） |
| `Alt + .` | 插入上一条命令的最后一个参数 |
| `Ctrl + T` | 交换光标处和前一个字符 |
| `Alt + T` | 交换光标处和前一个单词 |

---

## 3. 历史命令

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + P` | 上一条命令（同 `↑`） |
| `Ctrl + N` | 下一条命令（同 `↓`） |
| `Ctrl + R` | 反向搜索历史（再按继续搜） |
| `Ctrl + S` | 正向搜索历史 |
| `Ctrl + G` | 取消搜索 |
| `Ctrl + O` | 执行当前找到的命令并显示下一条 |
| `Alt + <` | 历史第一条 |
| `Alt + >` | 历史最后一条 |

### 历史命令扩展

| 符号 | 作用 |
|------|------|
| `!!` | 执行上一条命令 |
| `!n` | 执行历史第 n 条 |
| `!string` | 执行最近以 string 开头的命令 |

## 4. 复制粘贴

> **注意**：终端里的 `Ctrl+C/V` 和 GUI 不同，`Ctrl+C` 是中断程序，复制必须用 `Ctrl+Shift+C` 或鼠标选中自动复制。

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + Shift + C` | 复制（终端中） |
| `Ctrl + Shift + V` | 粘贴（终端中） |
| `Ctrl + Insert` | 复制 |
| `Shift + Insert` | 粘贴 |
| `Ctrl + C` | 发送中断信号（终止当前进程） |
| `Ctrl + V` | 字面输入下一个字符 |

## 5. 屏幕控制

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + L` | 清屏（同 `clear`） |
| `Ctrl + S` | 暂停屏幕输出 |
| `Ctrl + Q` | 恢复屏幕输出 |
| `Ctrl + Z` | 挂起当前进程（放入后台） |
| `fg` | 恢复挂起的进程到前台 |
| `bg` | 让挂起的进程在后台继续运行 |
| `jobs` | 查看后台任务列表 |

---

## 6. Tab 补全

| 快捷键 | 作用 |
|--------|------|
| `Tab` | 补全命令/文件名 |
| `Tab Tab` | 显示所有候选 |
| `Alt + ?` | 显示所有补全候选 |
| `Alt + *` | 插入所有匹配的文件名 |
| `Alt + /` | 只补全文件名 |
| `Alt + ~` | 只补全用户名 |
| `Alt + $` | 只补全变量名 |

---

## 7. 进程控制

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + C` | 终止前台进程 |
| `Ctrl + \` | 强制终止（产生 core dump） |
| `Ctrl + D` | 发送 EOF（关闭输入，退出 shell） |
| `Ctrl + Z` | 挂起（暂停）进程 |

---

## 8. tmux 多窗口

前缀键默认是 `Ctrl + B`，按完前缀再按对应键：

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + B, C` | 新建窗口 |
| `Ctrl + B, N` | 下一个窗口 |
| `Ctrl + B, P` | 上一个窗口 |
| `Ctrl + B, %` | 左右分屏 |
| `Ctrl + B, "` | 上下分屏 |
| `Ctrl + B, O` | 切换面板 |
| `Ctrl + B, D` | 分离会话（detach） |
| `tmux attach` | 重新连接会话 |

---

## 9. 快速记忆口诀

| 组合 | 含义 |
|------|------|
| `Ctrl + A/E` | **A**head / **E**nd |
| `Ctrl + B/F` | **B**ack / **F**orward |
| `Ctrl + U/K` | **U**pstream / **K**eep（向下删） |
| `Ctrl + W` | **W**ord（删单词） |
| `Ctrl + Y` | **Y**ank（粘贴） |
| `Ctrl + R` | **R**everse search |

## 不同 Shell 的差异

| Shell | 说明 |
|-------|------|
| **Bash** | 以上全部适用 |
| **Zsh** | 默认相同，可配置更多 |
| **Fish** | 部分不同，自带更友好的补全 |
