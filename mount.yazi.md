## 挂载
| Key binding | Alternate key | Action                |
| ----------- | ------------- | --------------------- |
| M           | -             | 打开挂载页面                |
| q           | -             | Quit the plugin       |
| k           | ↑             | Move up               |
| j           | ↓             | Move down             |
| l           | →             | Enter the mount point |
| m           | -             | Mount the partition   |
| u           | -             | Unmount the partition |
| e           | -             | Eject the disk        |
