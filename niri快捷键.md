| Hotkey                                          | 说明                                          |
| ----------------------------------------------- | ------------------------------------------- |
| Mod+Shift+/                                     | 显示重要 niri 快捷键列表                             |
| Mod+T                                           | 启动 alacritty（终端）                            |
| Mod+D                                           | 启动 fuzzel（应用启动器）                            |
| Super+Alt+L                                     | 启动 swaylock（屏幕锁）                            |
| Mod+Q                                           | 关闭当前聚焦窗口                                    |
| Mod+H 或 Mod+←                                   | 聚焦左侧列                                       |
| Mod+L 或 Mod+→                                   | 聚焦右侧列                                       |
| Mod+J 或 Mod+↓                                   | 聚焦列中下方窗口                                    |
| Mod+K 或 Mod+↑                                   | 聚焦列中上方窗口                                    |
| Mod+Ctrl+H 或 Mod+Ctrl+←                         | 将当前聚焦列向左移动                                  |
| Mod+Ctrl+L 或 Mod+Ctrl+→                         | 将当前聚焦列向右移动                                  |
| Mod+Ctrl+J 或 Mod+Ctrl+↓                         | 将当前聚焦窗口在列中向下移动                              |
| Mod+Ctrl+K 或 Mod+Ctrl+↑                         | 将当前聚焦窗口在列中向上移动                              |
| Mod+Shift+H/J/K/L 或 Mod+Shift+←/↓/↑/→           | 聚焦侧边显示器                                     |
| Mod+Ctrl+Shift+H/J/K/L 或 Mod+Ctrl+Shift+←/↓/↑/→ | 将当前聚焦列移动到侧边显示器                              |
| Mod+U 或 Mod+PageDown                            | 切换到下方工作区                                    |
| Mod+I 或 Mod+PageUp                              | 切换到上方工作区                                    |
| Mod+Ctrl+U 或 Mod+Ctrl+PageDown                  | 将当前聚焦列移动到下方工作区                              |
| Mod+Ctrl+I 或 Mod+Ctrl+PageUp                    | 将当前聚焦列移动到上方工作区                              |
| Mod+Shift+U 或 Mod+Shift+PageDown                | 将当前工作区向下移动                                  |
| Mod+Shift+I 或 Mod+Shift+PageUp                  | 将当前工作区向上移动                                  |
| Mod+\[                                          | 将当前聚焦窗口向左合并或拆分                              |
| Mod+]                                           | 将当前聚焦窗口向右合并或拆分                              |
| Mod+R 和 Mod+Shift+R                             | 向前/向后切换预设列宽                                 |
| Mod+M                                           | 最大化窗口                                       |
| Mod+C                                           | 将列在视图中居中                                    |
| Mod+-                                           | 列宽减少 10%                                    |
| Mod+=                                           | 列宽增加 10%                                    |
| Mod+Shift+-                                     | 窗口高度减少 10%                                  |
| Mod+Shift+=                                     | 窗口高度增加 10%                                  |
| Mod+Ctrl+R                                      | 重置窗口高度为自动                                   |
| Mod+Shift+F                                     | 切换当前聚焦窗口全屏                                  |
| Mod+V                                           | 将当前聚焦窗口在浮动与平铺布局间切换                          |
| Mod+Shift+V                                     | 在浮动与平铺布局间切换焦点                               |
| PrtSc                                           | 区域截图。用鼠标选择截图区域，按 Space 保存，按 Escape 取消       |
| Alt+PrtSc                                       | 截取当前聚焦窗口，复制到剪贴板并保存至 ~/Pictures/Screenshots/ |
| Ctrl+PrtSc                                      | 截取当前显示器，复制到剪贴板并保存至 ~/Pictures/Screenshots/  |
| Mod+Shift+E 或 Ctrl+Alt+Delete                   | 退出 niri                                     |
