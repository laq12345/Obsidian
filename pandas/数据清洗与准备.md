## 缺失值

我来帮您将这个表格翻译成中文。

**表 7.1: NA 处理对象方法**

| 方法 | 描述 |
|------|------|
| **dropna** | 根据每个标签的值是否缺失数据来过滤轴标签，可设置不同的缺失数据容忍阈值。 |
| **fillna** | 用某个值填充缺失数据，或使用插值方法如 `"ffill"` 或 `"bfill"`。 |
| **isna** | 返回布尔值，指示哪些值为缺失/NA。 |
| **notna** | **isna** 的取反，对于非 NA 值返回 `True`，对于 NA 值返回 `False`。 |

---

**补充说明：**
- **dropna**: 删除包含缺失值的行/列
- **fillna**: 填充缺失值（`ffill` = 前向填充，`bfill` = 后向填充）
- **isna**: 检测缺失值（返回 True/False）
- **notna**: 检测非缺失值（与 isna 相反）

### 1. `dropna()` — 删除缺失值

**作用：** 删除包含 NaN 的行或列

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'A': [1, 2, np.nan, 4],
    'B': [5, np.nan, np.nan, 8],
    'C': [9, 10, 11, 12]
})

# 删除任何包含 NaN 的行（默认 axis=0）
df.dropna()

# 删除任何包含 NaN 的列
df.dropna(axis=1)

# 只删除"全部为 NaN"的行
df.dropna(how='all')

# 保留至少有 2 个非 NaN 值的行
df.dropna(thresh=2)

# 只检查指定列是否缺失
df.dropna(subset=['A', 'B'])
```

| 参数 | 说明 |
|------|------|
| `axis=0` | 按行删除（默认） |
| `axis=1` | 按列删除 |
| `how='any'` | 任一值为 NaN 即删除（默认） |
| `how='all'` | 全部值为 NaN 才删除 |
| `thresh=n` | 至少保留 n 个非空值 |
| `subset=[...]` | 只在指定列中检查 |
| `inplace=True` | 原地修改 |

---

### 2. `fillna()` — 填充缺失值

**作用：** 用指定值或方法填充 NaN

```python
df = pd.DataFrame({
    'A': [1, 2, np.nan, 4],
    'B': [np.nan, np.nan, 7, 8]
})

# 用固定值填充
df.fillna(0)           # 全部填 0
df.fillna(-999)        # 全部填 -999

# 不同列用不同值填充
df.fillna({'A': 0, 'B': -1})

# 前向填充（ffill）：用前一个有效值填充
df.fillna(method='ffill')
# 等价写法: df.ffill()

# 后向填充（bfill）：用后一个有效值填充
df.fillna(method='bfill')
# 等价写法: df.bfill()

# 限制连续填充次数
df.fillna(method='ffill', limit=1)

# 用均值/中位数填充
df['A'].fillna(df['A'].mean())
df['B'].fillna(df['B'].median())

# 用插值方法填充（数值型）
df.interpolate()       # 线性插值
```

| 参数 | 说明 |
|------|------|
| `value` | 用于填充的标量、字典或 Series |
| `method='ffill'` | 前向填充 |
| `method='bfill'` | 后向填充 |
| `limit` | 最大连续填充次数 |
| `inplace=True` | 原地修改 |

#### 版本变更总结

| 功能 | 旧版 (Pandas < 2.0) | 新版 (Pandas ≥ 2.0) |
|------|---------------------|---------------------|
| 前向填充 | `df.fillna(method='ffill')` | **`df.ffill()`** |
| 后向填充 | `df.fillna(method='bfill')` | **`df.bfill()`** |
| 固定值填充 | `df.fillna(0)` | `df.fillna(0)` ✅ 不变 |
| 字典填充 | `df.fillna({'A':0})` | `df.fillna({'A':0})` ✅ 不变 |

---

### 3. `isna()` — 检测缺失值

**作用：** 返回布尔 DataFrame/Series，NaN 位置为 True

```python
df = pd.DataFrame({
    'A': [1, 2, np.nan, 4],
    'B': ['a', None, 'c', 'd']
})

# 返回布尔矩阵
df.isna()
# 输出:
#        A      B
# 0  False  False
# 1  False   True
# 2   True  False
# 3  False  False

# 统计每列的缺失值数量
df.isna().sum()
# A    1
# B    1

# 统计整个 DataFrame 的缺失总数
df.isna().sum().sum()   # 2

# 筛选出包含缺失值的行
df[df.isna().any(axis=1)]

# 筛选出全部非空的行
df[df.notna().all(axis=1)]
```

> **注意：** `isnull()` 是 `isna()` 的别名，功能完全相同。

---

### 4. `notna()` — 检测非缺失值

**作用：** 与 `isna()` 相反，非 NaN 位置为 True

```python
df = pd.DataFrame({
    'A': [1, np.nan, 3],
    'B': [np.nan, np.nan, 6]
})

df.notna()
# 输出:
#        A      B
# 0   True  False
# 1  False  False
# 2   True   True

# 统计每列的有效值数量
df.notna().sum()
# A    2
# B    1

# 实际应用：筛选某列不为空的记录
df[df['A'].notna()]
```

> **注意：** `notnull()` 是 `notna()` 的别名，功能完全相同。

---

### 方法对比速查表

| 场景 | 推荐方法 |
|------|----------|
| 清洗数据，直接丢弃含缺失的行 | `df.dropna()` |
| 将缺失值替换为默认值（如 0） | `df.fillna(0)` |
| 时间序列数据填补空缺 | `df.ffill()` 或 `df.bfill()` |
| 查看数据哪里有缺失 | `df.isna().sum()` |
| 筛选有效数据行 | `df[df['col'].notna()]` |
| 数值数据插值 | `df.interpolate()` |

这四个方法是 Pandas 数据清洗中最基础也最常用的工具！

## Pandas 重复值处理方法：`duplicated()` 和 `drop_duplicates()`

### 一、`duplicated()` — 标记重复行

**作用：** 返回布尔 Series，标记哪些行是重复的（重复行返回 `True`）

```python
import pandas as pd

df = pd.DataFrame({
    'name': ['Alice', 'Bob', 'Alice', 'Charlie', 'Bob'],
    'age': [25, 30, 25, 35, 30],
    'city': ['Beijing', 'Shanghai', 'Beijing', 'Guangzhou', 'Shanghai']
})

# 检查完全重复的行
df.duplicated()
# 输出:
# 0    False   ← 第一次出现，不算重复
# 1    False
# 2     True   ← 与第0行重复
# 3    False
# 4     True   ← 与第1行重复

# 统计重复行数量
df.duplicated().sum()          # 2

# 基于指定列判断重复
df.duplicated(subset=['name'])
df.duplicated(subset=['name', 'age'])

# 标记"第一次出现"为重复（保留最后一个）
df.duplicated(keep='last')
# 输出:
# 0     True   ← 第0行与第2行重复，保留第2行
# 1     True   ← 第1行与第4行重复，保留第4行
# 2    False
# 3    False
# 4    False

# 标记所有重复项（首次和后续都标为True）
df.duplicated(keep=False)
# 输出:
# 0     True
# 1     True
# 2     True
# 3    False
# 4     True
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `subset` | 指定判断重复的列（列表） | 全部列 |
| `keep='first'` | 首次出现标记为 **False**（不重复），后续标记为 **True** | `'first'` |
| `keep='last'` | 最后一次出现标记为 **False**，之前的标记为 **True** | - |
| `keep=False` | 所有重复行都标记为 **True** | - |

---

### 二、`drop_duplicates()` — 删除重复行

**作用：** 直接删除重复行，返回去重后的 DataFrame

```python
df = pd.DataFrame({
    'name': ['Alice', 'Bob', 'Alice', 'Charlie', 'Bob'],
    'age': [25, 30, 25, 35, 30],
    'city': ['Beijing', 'Shanghai', 'Beijing', 'Guangzhou', 'Shanghai']
})

# 删除完全重复的行（默认保留第一个）
df.drop_duplicates()
# 结果: 保留第0,1,3行，删除第2,4行

# 基于指定列去重
df.drop_duplicates(subset=['name'])
# 结果: 每个 name 只保留第一条记录

df.drop_duplicates(subset=['name', 'age'])

# 保留最后一条记录
df.drop_duplicates(keep='last')
# 结果: 保留第2,3,4行，删除第0,1行

# 原地修改
df.drop_duplicates(inplace=True)

# 删除重复后重置索引
df.drop_duplicates().reset_index(drop=True)
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `subset` | 基于哪些列判断重复 | 全部列 |
| `keep='first'` | 保留第一次出现的行 | `'first'` |
| `keep='last'` | 保留最后一次出现的行 | - |
| `inplace=True` | 原地修改，不返回新对象 | `False` |
| `ignore_index=True` | 重置索引（Pandas 1.0+） | `False` |

---

### 三、实际应用场景示例

#### 场景 1：数据清洗 — 去除完全重复的记录

```python

```

                      '2024-01-03', '2024-01-05', '2024-01-06']
})

# 按 user_id 去重，保留最新注册的记录

## Pandas `replace()` 方法详解

### 一、基本用法

**作用：** 将 DataFrame/Series 中的值替换为其他值

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'A': [1, 2, 3, 4, 5],
    'B': ['a', 'b', 'c', 'b', 'a'],
    'C': [10, -999, 20, -999, 30]
})
```

---

### 二、标量替换 — 单值替换单值

```python
# 替换单个值
df.replace(1, 100)           # 把所有 1 替换为 100
df.replace('a', 'apple')     # 把所有 'a' 替换为 'apple'
df.replace(-999, np.nan)     # 把 -999（缺失值标记）替换为 NaN
```

---

### 三、字典替换 — 多值映射替换 ⭐ 最常用

```python
# 用字典同时替换多个值
df.replace({1: 100, 2: 200, 3: 300})

# 字符串替换
df.replace({'a': 'alpha', 'b': 'beta', 'c': 'gamma'})

# 实际场景：数据清洗，将异常值/缺失值标记统一处理
df = pd.DataFrame({
    'score': [95, -1, 88, -1, 72, -999],
    'status': ['active', 'N/A', 'active', 'null', 'pending', 'N/A']
})

# 统一替换各种"缺失值"表示
df.replace({
    -1: np.nan,
    -999: np.nan,
    'N/A': np.nan,
    'null': np.nan
})
```

---

### 四、按列指定不同的替换规则

```python
df = pd.DataFrame({
    'city': ['Beijing', 'Shanghai', 'Beijing', 'Guangzhou'],
    'code': ['BJ', 'SH', 'BJ', 'GZ'],
    'level': ['A', 'B+', 'A', 'B']
})

# 不同列使用不同的替换字典
df.replace({
    'city': {'Beijing': '北京', 'Shanghai': '上海', 'Guangzhou': '广州'},
    'level': {'A': '优秀', 'B+': '良好', 'B': '合格'}
})
```

结果：
```
      city code level
0      北京   BJ   优秀
1      上海   SH   良好
2      北京   BJ   优秀
3      广州   GZ   合格
```

---

### 五、正则表达式替换 🔥 高级用法

```python
s = pd.Series(['foo-123', 'bar-456', 'baz-789', 'error'])

# 使用正则表达式
s.replace(r'\d+', '', regex=True)
# 结果: ['foo-', 'bar-', 'baz-', 'error']

# 正则 + 替换值
s.replace(r'[a-z]+', 'X', regex=True)
# 结果: ['X-123', 'X-456', 'X-789', 'X']

# 更复杂的例子：清洗电话号码格式
phones = pd.Series(['138-1234-5678', '(139)12345678', '150 9876 5432'])
phones.replace(r'[\-\(\)\s]', '', regex=True)
# 结果: ['13812345678', '13912345678', '15098765432']
```

---

### 六、常用参数一览

| 参数 | 说明 | 示例 |
|------|------|------|
| `to_replace` | 要查找的值（标量/列表/字典/正则） | `1`, `[1,2]`, `{'a':'A'}` |
| `value` | 替换后的值 | `0`, `np.nan` |
| `regex=True` | 启用正则表达式匹配 | 见上方示例 |
| `inplace=True` | 原地修改 | - |
| `limit=n` | 限制每列最多替换次数 | `limit=1` |

### 七、实际应用场景合集

#### 场景 1：数据标准化

```python
df = pd.DataFrame({
    'gender': ['M', 'F', 'male', 'female', 'm', 'f', 'Male']
})

# 统一性别字段
df['gender'] = df['gender'].replace({
    'M': '男', 'male': '男', 'm': '男', 'Male': '男',
    'F': '女', 'female': '女', 'f': '女'
})
```

#### 场景 2：处理异常值

```python
scores = pd.DataFrame({
    'math': [85, 120, 92, -5, 78],
    'english': [90, 88, 305, 76, 82]
})

# 分数超出合理范围的替换为 NaN
scores.replace({r'^[3-9]\d{2}$|^1[2-9]\d$|^[2-9]\d{2}$': np.nan}, 
               regex=True)

# 或者用范围方式
import numpy as np
scores = scores.mask((scores < 0) | (scores > 100))
```

#### 场景 3：DataFrame 级别的批量替换

```python
df = pd.DataFrame({
    'col1': [1, 2, 3, None],
    'col2': ['yes', 'no', 'yes', 'unknown'],
    'col3': [0.1, 0.2, None, 0.4]
})

# 一条语句完成多种替换
clean_df = df.replace({
    None: np.nan,          # None → NaN
    'unknown': np.nan,     # unknown → NaN  
    'yes': True,           # yes → True
    'no': False            # no → False
})
```

### 八、`replace()` vs 其他方法对比

| 需求 | 推荐方法 |
|------|----------|
| 缺失值填充 | **`fillna()`** ✅ 更高效 |
| 值的映射替换 | **`replace()`** ✅ 最灵活 |
| 批量数值运算 | `apply()` / 向量化运算 |
| 列名重命名 | `rename()` |
| 数据类型转换 | `astype()` |

> 💡 **性能提示：** 对于大量数据的简单替换，`replace()` 比 `apply()` 更快；对于纯缺失值填充，`fillna()` 比 `replace(np.nan)` 更高效。

## Pandas `rename()` 方法笔记总结

### 一、核心语法

```python
df.rename(columns={'旧列名': '新列名'})    # 重命名列
df.rename(index={'旧行索引': '新行索引'})   # 重命名索引
```

---

### 二、重命名列名（columns）

```python
# 单个列
df.rename(columns={'name': '姓名'})

# 多个列
df.rename(columns={
    'name': '姓名',
    'age': '年龄',
    'city': '城市'
})

# 只改部分，其他不变
df.rename(columns={'name': '员工姓名'})
```

---

### 三、重命名索引（index）

```python
df.rename(index={'r1': 'row_1', 'r2': 'row_2'})

df.rename(index={
    'r1': '员工A',
    'r2': '员工B',
    'r3': '员工C'
})
```

---

### 四、同时改 columns 和 index

```python
df.rename(
    columns={'name': '姓名', 'age': '年龄'},
    index={'r1': 'id_001', 'r2': 'id_002'}
)
```

---

### 五、函数批量重命名 🔥 最实用

```python
# 全部大写
df.rename(columns=str.upper)

# 全部小写
df.rename(columns=str.lower)

# 首字母大写
df.rename(columns=str.capitalize)

# 加前缀
df.rename(columns=lambda x: f'col_{x}')

# 空格→下划线（清洗脏数据）
df.rename(columns=lambda x: x.replace(' ', '_'))

# 去空格 + 小写 + 空格变下划线
df.rename(columns=lambda x: x.strip().lower().replace(' ', '_'))
```

---

### 六、axis 参数简写

```python
df.rename({'name': '姓名'}, axis=1)       # axis=1 → 列
df.rename({'r1': '第一行'}, axis=0)      # axis=0 → 行
```

### 八、实战场景速查

### 九、与同类方法对比

| 需求 | 方法 |
|------|------|

## `pd.cut()` 和 `df.sample()` 常用用法笔记

---

### 一、`pd.cut()` — 连续值分箱/离散化

**核心：把数值切成区间，打上标签。**

#### 1. 等宽分箱（自动均分）

```python
pd.cut(data, bins=3)
# 自动将 (min, max) 均分成 3 段
```

#### 2. 自定义边界 ⭐ 最常用

```python
pd.cut(ages, bins=[0, 18, 35, 60, 100])
# 输出区间: (0,18], (18,35], (35,60], (60,100]
```

#### 3. 自定义边界 + 标签 🔥 实战必备

```python
pd.cut(
    df['age'],
    bins=[0, 18, 35, 60, 100],
    labels=['青少年', '青年', '中年', '老年']
)
# 输出: 青少年 / 青年 / 中年 / 老年
```

#### 4. 配合 DataFrame 新增列

```python
df['等级'] = pd.cut(df['score'], bins=[0, 60, 75, 90, 100],
                    labels=['不及格', '及格', '良好', '优秀'])
```

#### 5. 统计各箱数量

```python
pd.cut(scores, bins=4).value_counts()           # 各箱计数
pd.cut(scores, bins=4).value_counts().sort_index()  # 按顺序排列
```

#### 6. 分组聚合分析

```python
df.groupby(pd.cut(df['age'], bins=[0,30,50,100]))['salary'].mean()
```

#### 7. 重要参数速查

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `bins` | 整数=等宽n箱，列表=自定义边界 | 必填 |
| `labels` | 区间标签名称 | 显示数值区间 |
| `right=True` | 左开右闭 `(a,b]` | True |
| `include_lowest` | 包含最左端点 | False |

#### 8. cut vs qcut

```python
pd.cut(data, bins=4)   # 等宽（每段宽度相同）
pd.qcut(data, q=4)     # 等频（每段样本数大致相同）
```

### 二、`df.sample()` — 随机抽样

**核心：从数据中随机抽取行或列。**

#### 1. 抽固定数量

```python
df.sample(n=5)                      # 随机抽5行
df.sample(n=5, random_state=42)     # 可复现（结果固定）
```

#### 2. 按比例抽 ⭐ 常用

```python
df.sample(frac=0.3)                 # 随机抽30%
df.sample(frac=1, random_state=42)  # = 随机打乱全部顺序
```

#### 3. 有放回抽样 🔥

```python
df.sample(n=30, replace=True)       # 可重复抽取同一行
df.sample(frac=2, replace=True)     # 抽取超过100%
```
> **用途：** Bootstrap 重采样估计置信区间

#### 4. 加权抽样 🔥 高级

```python
df.sample(n=5, weights=df['salary'])       # 薪资越高越易被选中
df.sample(n=5, weights=df['city'].map({'北京':3,'上海':1}))  # 自定义权重
```

#### 5. 其他选项

```python
df.sample(n=3, axis=1)                   # 按列随机抽取
df.sample(n=5, ignore_index=True)        # 抽完重置索引为0,1,2...
```

#### 6. 划分训练/测试集 🤖 最常用模式

```python
# 方法一：分别抽样
train = df.sample(frac=0.8, random_state=42)
test  = df.drop(train.index)

# 方法二：分层抽样（保持各类比例一致）
train = df.groupby('类别').sample(frac=0.8, random_state=42)
test  = df.drop(train.index)

# 方法三：先打乱再切分
shuffled = df.sample(frac=1, random_state=42)
train = shuffled.iloc[:int(len(df)*0.8)]
test  = shuffled.iloc[int(len(df)*0.8):]
```

#### 7. Series 抽样（抽奖场景）

```python
names = pd.Series(['张三','李四','王五','赵六','钱七'])
names.sample(n=3, random_state=7)            # 抽3个中奖者
names.sample(n=5, replace=True, random_state=7)  # 有放回抽奖
```

|------|-------|----------|

## Pandas扩展数据类型

## 表 7.3：Pandas 扩展数据类型

| 扩展类型 | 描述 |
|---------|------|
| **BooleanDtype** | 可空布尔数据，传字符串时使用 `"boolean"` |
| **CategoricalDtype** | 分类数据类型，传字符串时使用 `"category"` |
| **DatetimeTZDtype** | 带时区的日期时间 |
| **Float32Dtype** | 32位可空浮点数，传字符串时使用 `"Float32"` |
| **Float64Dtype** | 64位可空浮点数，传字符串时使用 `"Float64"` |
| **Int8Dtype** | 8位可空有符号整数，传字符串时使用 `"Int8"` |
| **Int16Dtype** | 16位可空有符号整数，传字符串时使用 `"Int16"` |
| **Int32Dtype** | 32位可空有符号整数，传字符串时使用 `"Int32"` |
| **Int64Dtype** | 64位可空有符号整数，传字符串时使用 `"Int64"` |
| **UInt8Dtype** | 8位可空无符号整数，传字符串时使用 `"UInt8"` |
| **UInt16Dtype** | 16位可空无符号整数，传字符串时使用 `"UInt16"` |
| **UInt32Dtype** | 32位可空无符号整数，传字符串时使用 `"UInt32"` |
| **UInt64Dtype** | 64位可空无符号整数，传字符串时使用 `"UInt64"` |

---

### 核心要点

这些就是 Pandas 的 **Nullable Extension Types（可空扩展类型）**，它们的共同特点是：

- ✅ **支持 `pd.NA` 缺失值**（不会像 numpy 类型那样把 NaN 静默转成其他值）
- 🔤 **传字符串时的简写名**就是 Description 中引号里的内容

```python
# 三种写法等价：
pd.array([1, 2, None], dtype=pd.Int64Dtype())    # 完整类名
pd.array([1, 2, None], dtype="Int64")             # 字符串简写 ⭐ 最常用
pd.Series([1, 2, None], dtype="Int64")            # Series 中直接用
```
