pandas.read_csv函数的参数

### 将数据写入文本格式

`to_csv` 是 Pandas 中最常用的 I/O 方法之一，用于将 DataFrame 或 Series 保存为 CSV 文件。为了方便你记笔记，我将其整理为 **核心语法**、**常用参数详解** 和 **实战示例** 三个部分。

---

### 📝 `to_csv` 核心语法

```python
DataFrame.to_csv(
    path_or_buf=None, 
    sep=',', 
    na_rep='', 
    columns=None, 
    header=True, 
    index=True, 
    index_label=None, 
    mode='w', 
    encoding=None, 
    date_format=None, 
    quoting=None
)
```

---

### 🔑 常用参数详解

#### 1. 文件路径与基本格式

- **`path_or_buf`**：
    - **描述**：文件路径（字符串）或文件句柄。
    - **注意**：如果设为 `None`，则返回字符串形式的数据，不写入文件。
- **`sep`**：
    - **描述**：字段的分隔符，默认是逗号 `','`。
    - **常用**：制表符 `'\t'` 常用于生成 TSV 文件。
- **`na_rep`**：
    - **描述**：用于替换缺失值（NaN）的字符串。
    - **默认**：空字符串 `''`。

#### 2. 索引与表头控制

- **`index`**：
    - **描述**：是否将行索引写入文件。
    - **默认**：`True`。
    - **建议**：如果索引没有实际意义（如默认的 0, 1, 2...），通常设为 `False`。
- **`header`**：
    - **描述**：是否将列名写入文件。
    - **默认**：`True`。
- **`columns`**：
    - **描述**：指定要写入的列的子集（列表形式）。

#### 3. 写入模式与编码

- **`mode`**：
    - **描述**：Python 的文件写入模式。
    - **默认**：`'w'` (覆盖写入)。
    - **常用**：`'a'` (追加写入，常用于日志或分块写入)。
- **`encoding`**：
    - **描述**：文件的字符编码。
    - **常用**：`'utf-8'` (标准)，`'gbk'` 或 `'utf-8-sig'` (Excel 打开中文不乱码常用)。

#### 4. 高级格式控制

- **`date_format`**：
    - **描述**：日期时间对象的格式化字符串（如 `'%Y-%m-%d'`）。
- **`quoting`**：
    - **描述**：控制引号的行为。
    - **常用**：`csv.QUOTE_MINIMAL` (默认，仅必要时加引号)，`csv.QUOTE_ALL` (所有字段都加引号)。
- **`float_format`**：
    - **描述**：浮点数的格式化字符串（如 `'%.2f'` 保留两位小数）。

---

### 💡 实战示例

#### 1. 最基础的保存

```python
df.to_csv('data/output.csv')
```

#### 2. 去掉索引，使用制表符分隔

```python
# 适合用于生成没有杂乱索引列的干净数据
df.to_csv('data/clean_data.tsv', sep='\t', index=False)
```

#### 3. 解决 Excel 中文乱码问题

```python
# 使用 utf-8-sig 编码，Excel 能正确识别 UTF-8
df.to_csv('data/chinese_data.csv', encoding='utf-8-sig', index=False)
```

#### 4. 追加数据到现有文件

```python
# 第一次写入（包含表头）
df1.to_csv('data/log.csv', mode='w', header=True, index=False)

# 追加写入（不包含表头，避免重复）
df2.to_csv('data/log.csv', mode='a', header=False, index=False)
```

#### 5. 格式化输出

```python
import csv
# 浮点数保留2位，空值显示为0，所有字段加引号
df.to_csv('data/formatted.csv', 
          float_format='%.2f', 
          na_rep='0', 
          quoting=csv.QUOTE_ALL)
```
